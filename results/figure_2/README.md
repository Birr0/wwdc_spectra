# Figure 2 — SDSS Main Galaxy Sample

Analysis and visualization of the SDSS main galaxy sample used in this work.

## Files

| File | Description |
|---|---|
| `sdss_main_galaxy_sample_v2.ipynb` | Produces the figure characterizing the galaxy sample |

## What it shows

Characterizes the SDSS galaxy sample: distributions of redshift, apparent magnitude, and other survey properties. Illustrates the selection effects present in the data and motivates the use of volume-limited and magnitude-limited subsamples for the disentanglement analysis.

## Reproducing the figure

Open `sdss_main_galaxy_sample_v2.ipynb` and run all cells. Requires the SDSS dataset to be available at `$DATA_ROOT/sdss/`.
